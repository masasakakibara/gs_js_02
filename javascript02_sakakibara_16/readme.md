①課題内容（どんな作品か）

授業で作成したメモパッドを再現し、そこに新しい機能を乗せることを目指した
本来は１）ランダム関数で出た結果が自動的に入力・Saveされる、２）さらにその関数を出した日付を出し、それを自動的にSaveしていく
という仕組みを作りたかったが、それぞれの結果だけ別々に出し、マニュアルで入れられるというところで終わった
